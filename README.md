# 🚨 Solana Whale Tracker (Lite)

A lightweight Python script to monitor high-value wallet movements on the Solana blockchain.

## 🛠️ Features
- Real-time polling of Solana Mainnet-Beta.
- Logs signature movements to the console.
- Zero-dependency logic.

## 🚀 Quick Start
1. Clone the repo.
2. Edit `target_wallet` in `main.py`.
3. Run `python main.py`.
